HOUND.Contact.Site:getTextData (utmZone, MGRSdigits)
return Information used in Text messages primary emitter

Parameters:

  • utmZone (bool) True will add UTM zone to response
  • MGRSdigits (Number) number of digits in the MGRS part of the response (eg. 2 = 12, 5=12345)

Returns:

    GridPos (string) MGRS grid position (eg. "CY 564 123", "DN 2 4") Return BE (string) Bullseye position string (eg. "035/15", "187/120")

  • MGRSdigits (Number) number of digits in the MGRS part of the response (eg. 2 = 12, 5=12345)

Returns:

    GridPos (string) MGRS grid position (eg. "Charlie Yankee one two Three four") Return BE (string) Bullseye position string (eg. "Zero Three Five 15")
